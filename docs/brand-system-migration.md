# Vivah Australia — Brand System Migration Reference

This document maps out the approved brand identity guidelines and migration path for the platform design system.

---

## 🎨 Visual Identity & Colors

The brand moves away from a generic SaaS appearance to feel like a premium, trustworthy matrimonial matchmaking partner.

| Category | Token | Hex Code | Purpose / Usage |
| :--- | :--- | :--- | :--- |
| **Primary Brand** | `Maroon` | `#A10E4D` | Primary CTAs, active highlights, navigation headers, vital links |
| **Secondary Brand** | `Wedding Gold` | `#D4A04C` | Premium badges, trust highlights, verification rings, success states |
| **Accent** | `Soft Rose` | `#E74C7C` | Hover transitions, highlights, focus visual highlights |
| **Background** | `Warm Ivory` | `#FFF9F5` | Main body backgrounds, dashboard cards, layout panels |
| **Text** | `Charcoal` | `#2F2F2F` | Primary text copy, body copy, descriptions |

---

## ✍️ Typography Hierarchy

1. **Primary Headings (`Playfair Display`):** Elegant, serif. Used for marketing heros, titles, main pages, landing sections.
2. **Secondary Headings (`Cormorant Garamond`):** Light, graceful serif. Used for card titles, section headers, biodata subtitles.
3. **Body Text (`Poppins`):** Modern, geometric sans-serif. Used for form inputs, navigation links, and standard body text.

---

## ♿ Accessibility Guidelines

- Maintain a minimum contrast ratio of **4.5:1** for normal text and **3:1** for large text to hit WCAG AA standards.
- Maroon (`#A10E4D`) on Warm Ivory (`#FFF9F5`) exceeds **7:1** and is fully compliant.
- Wedding Gold (`#D4A04C`) should NOT be used for primary text; restrict its use to badges, icons, and decorative borders.
